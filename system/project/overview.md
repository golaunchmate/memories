# LaunchMate

Platform for SDSU's ZIP Launchpad startup founders using Letta AI agents for mentorship and task management.

## Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS 4
- **Backend**: Supabase (PostgreSQL), Clerk (auth)
- **AI**: Letta SDK for agent orchestration
